# Arthmetic Calculator

This is a small calculator project which parses the input and using AST(Abstract syntax tree) 


Supported Operation:

| Operation Name | Symbol |
|----------------|--------|
| Add            | `+`    |
| Subtract       | `-`    |
| Multiply       | `*`    |
| Divide         | `/`    |
| PowerOf        | `^`    |


## Building from scratch

```
cargo build
```

### if you want to build for release

```
cargo build --release
```
